Installation guide

Install git - https://git-scm.com/book/en/v2/Getting-Started-Installing-Git
add git to path
Install java (Min version 11.0.0) - https://www.oracle.com/java/technologies/javase-jdk13-downloads.html
add java to path
install gradle following these instructions - https://gradle.org/install/
add gradle to path
run the command 'gradle -v' to check your java and gradle version
get the most recent version of the project either by cloning or pulling from the repository - https://github.com/georgefox6/FootballLeague.git
in the root FootballLeague directory run the command 'gradle run'
